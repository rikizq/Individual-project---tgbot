Для поиска музыки в боте используется iTunes Search API от Apple Inc. Это бесплатный и открытый API, который позволяет искать треки из каталога iTunes Store и Apple Music без регистрации и API-ключа. Запрос отправляется на адрес https://itunes.apple.com/search С параметрами term (поисковый запрос), entity=song (поиск песен), limit (количество результатов) и country=RU (регион). В ответ API возвращает JSON со списком треков, где содержатся название песни, исполнитель и ссылка previewUrl. previewUrl используется ботом для отправки официального 30-секундного аудиофрагмента в Telegram. Полные треки скачивать нельзя, разрешено использовать только preview.
